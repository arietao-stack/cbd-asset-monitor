    How we measure

    Honest, transparent, no invented numbers.
    VIEWS
    Views / impressions — pulled per post via yt-dlp. TikTok photo posts and some Instagram posts can't expose view counts.
    ENG
    Engagement = likes + comments + shares (TikTok) / likes + comments (Instagram). Eng % = engagement ÷ views.
    IG
    Instagram needs a logged-in session and blocks cloud IPs, so it's refreshed from a local machine; blocked posts keep their last-known value until the next run.
    CHANNEL
    Owned = Inhouse + Social Boost · Earned = KOL/UGC collab, derived from the tracker's Source Type.
